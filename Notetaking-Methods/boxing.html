Boxing Method 📒

Best For: Digital note-taking (iPad or other tablet)

Boxes are used to separate topics visually within a page. It was originally created with only digitally note-taking in mind, but can be adapted to handwritten notes fairly easily.

These notes are visually pleasing and very easy to use for studying or review. This method is a lot easier to be used digitally because it takes advantage of features such as resizing, moving, and geometry drawing.

Pros
  • Great for visual learners.
  • Notes are visually appealing.
  • Very easy to use.

Cons
  • Can be time-consuming.
  • Not always practical for live lectures, especially if they are faster paced.
